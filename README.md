<H1>Factory Health Dashboard</H1>

<H2>1. Project Title</H2>

🏭 Factory Health & Fairness Dashboard  


<H2>2. Short Description</H2>
 
An interactive Tableau dashboard providing insights into factory operations, machine health, and workforce equality using Deloitte Forage simulation data.


<H2>3. Tech Stack</H2>

• 📊 Tableau Desktop  
• 📂 Data Cleaning (Excel / CSV)  
• 📈 Calculated Fields & Aggregations  
• 📁 .twbx / .png  


<H2>4. Data Source</H2>

Source: Deloitte Australia Data Analytics Virtual Experience (Forage)  

Includes:

• Factory (Berlin, Shenzhen, etc.)  
• Job Role  
• Equality Score  
• Equality Class (Fair / Unfair / Highly Discriminative)  
• Device Type  
• Machine Health Status  


<H2>5. Features / Highlights</H2>

🔎 Business Problem

Manufacturing organizations often struggle to monitor machine health and ensure fairness across job roles due to fragmented and complex data, making it difficult to identify risks and take timely action.


<H2>🎯 Goal of the Dashboard</H2>

• Monitor unhealthy machines across factories  
• Identify high-risk device types  
• Analyze equality across job roles  
• Compare factory performance  
• Support operational and HR decision-making  


<H2>📊 Key Visuals</H2>

📌 Factory-wise Unhealthy Machines (Bar Chart)  
• Comparison of unhealthy devices across factories  

📊 Device Type Analysis (Bar Chart)  
• CNC, Conveyor, Furnace, Laser, Heavy Machines, etc.  

📋 Equality Score Table  
• Displays job roles with equality scores and classification  

📊 Equality Classification Insight  
• Fair vs Unfair vs Highly Discriminative distribution  


🎛️ Filters  
• Factory  
• Job Role  
• Device Type  
• Equality Class  


<H2>🚀 Business Impact</H2>

• Improves machine maintenance planning  
• Helps reduce downtime  
• Identifies high-risk equipment  
• Supports fair workplace analysis  
• Enables data-driven decision-making
